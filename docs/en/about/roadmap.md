# Roadmap & Platform Plan

This page summarizes the phased roadmap for HeritageGraph (from `PLATFORM_PLAN.md`).

Phases:

- Phase 1: Assertion-first architecture, multi-step forms, contributor onboarding
- Phase 2: Record pages, triaged reviewer workspace, improved UX
- Phase 3: Syncretic relationships, calendar conversions, mobile field surveys

For detailed milestones and implementation notes see `PLATFORM_PLAN.md` in the repository.

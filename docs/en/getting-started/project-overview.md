# Project Overview

HeritageGraph preserves and publishes cultural heritage as linked open data. This page gives a short overview for newcomers.

Key goals:
- Enable community contributions via web forms
- Provide a moderation & epistemic review workflow
- Publish data as linked open data aligned to CIDOC-CRM

See `ARCHITECTURE.md`, `PLATFORM_PLAN.md`, and `AGENTS.md` in the repository for the full vision and roadmap.
